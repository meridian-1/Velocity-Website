# Velocity
https://meridian-1.github.io/Velocity-Website/

Velocity is a modern, responsive web landing page designed for a design agency or creative company. It showcases services like Graphic Design, Web Design, and UI/UX Design through an elegant and clean interface.

## Features

- Responsive design that works on all devices
- Clean and modern UI with custom fonts
- Sections for welcome, about, services, and contact
- Optimized for performance with efficient CSS

## Project Structure

```
Velocity/
├── index.html          # Main HTML file
├── style.css           # Stylesheet with custom fonts and responsive design
├── js/
│   └── main.js         # Main JavaScript file (currently empty)
├── fonts/              # Custom fonts (Open Sans)
├── icons/              # Icon assets
└── images/             # Image assets
```

## Customization
- **Styles**: Edit `style.css` to modify colors, fonts, and layout.
- **Content**: Update `index.html` to change text, images, and links.

## Technologies Used
- HTML5
- CSS3
- JavaScript
